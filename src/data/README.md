# Data Files Documentation

## orders.json
Contains mock customer orders data. Each order includes:
- Customer information (name, email, shipping address)
- Order details (ID, total, date)
- Line items (gift boxes ordered)

## box_products.json
Maps each gift box type to its individual products:
- Valentine Box → roses, chocolates, card, perfume
- Birthday Box → cupcake, gift card, birthday card
- Client Gift Box → wine, fruit basket, pen

These files work together to help generate the warehouse picking list.
